📌 Sobre Este Projeto

Este projeto apresenta uma análise completa e interativa da infraestrutura logística de Santa Catarina, integrando dados oficiais do IBGE (BC25 - 2020). O mapa inclui 14 camadas temáticas com 10.250+ elementos geográficos, oferecendo visualização profissional e análise estatística detalhada.

⚡ Recursos Principais

  • 14 camadas temáticas
  • Controle de visibilidade
  • Pop-ups informativos
  • Zoom e navegação fluida
  • Compass de orientação

📈 Relatório Estatístico

  • 11 abas temáticas
