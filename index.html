📌 Sobre Este Projeto

Este projeto apresenta uma análise completa e interativa da infraestrutura logística de Santa Catarina, integrando dados oficiais do IBGE (BC25 - 2020). O mapa inclui 14 camadas temáticas com 10.250+ elementos geográficos, oferecendo visualização profissional e análise estatística detalhada.

🎨 Como Usar o Gerador de Mapas Customizados

O Gerador permite criar mapas personalizados com apenas as camadas que você precisa!

📝 Passo a Passo:

  1. Acesse o Gerador: Clique no botão "🎨 Gerador de Mapas Customizados"
  2. Selecione Camadas: Marque as camadas desejadas nas categorias:
    • 🚗 Transportes: Rodovias Federais, Estaduais, Ferrovias
    • 🌉 Obras de Arte: Pontes, Túneis, Viadutos
    • ⚓ Marítima: Hidrovias, Dutos, Terminais Portuários
    • ✈️ Aviação: Helipontos, Construções Aeroportuárias
    • 🗺️ Limites: Limite Estadual, Municípios
  3. Dê um Nome: Digite um nome para seu mapa (opcional)
  4. Gere o Mapa: Clique em "Gerar Mapa" e aguarde ~5-10 segundos
  5. Visualize: O mapa aparecerá na tela com as camadas selecionadas
  6. Exporte PNG: Clique em "Exportar PNG" para baixar uma imagem em alta qualidade
💡 Dica: Selecione apenas as camadas que precisa para ter um mapa mais limpo e focado. Combine diferentes camadas para análises específicas!
