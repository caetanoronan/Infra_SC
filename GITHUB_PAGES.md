# GitHub Pages Configuration

Este arquivo configura automaticamente o GitHub Pages para servir este projeto.

## Como Ativar GitHub Pages

1. **Vá para as configurações do repositório**
   - Clique em "Settings" no seu repositório GitHub

2. **Localize GitHub Pages**
   - Procure pela seção "Pages" no menu lateral esquerdo

3. **Configure a Source**
   - Em "Source", selecione a branch: `main`
   - Mantenha o diretório como `/ (root)`

4. **Acesse seu site**
   - GitHub criará uma URL como: `https://seu-usuario.github.io/nome-repositorio`
   - O site estará disponível em poucas segundos

## Recursos do GitHub Pages

✅ Hospedagem **gratuita** e **ilimitada**
✅ HTTPS automático e seguro
✅ Domínio personalizado (opcional)
✅ Sem tempo de inatividade

## Dicas

- Se fizer alterações, aguarde alguns segundos para o site atualizar
- O cache pode ser limpo forçando F5 ou Ctrl+Shift+Delete
- Compartilhe a URL com qualquer pessoa - é pública!
